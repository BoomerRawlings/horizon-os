# Horizon OS Vault Guide

This vault is the durable, human-readable memory layer for Horizon. Keep it boring, searchable,
and small. Horizon is the app; this folder is the database.

## Capture Rules

- Raw captures go to `Inbox/Captures/`.
- Items waiting to be sorted go to `Inbox/To Triage/`.
- If a capture has a date, it becomes an item in `Calendar/Items/`.
- If a capture is intentionally undated, keep it as an open reminder or inbox note.

## Write Rules

- Do not invent dates, categories, or relationships.
- Keep secrets out of Markdown notes.
- Prefer updating an existing index over adding a new organizing system.
- Avoid destructive edits unless you clearly intend them.

Full, feature-by-feature help lives inside the app under **Settings → Advanced → How to use Horizon**.
