Set up Horizon

Set up the vault in Obsidian first. Horizon then connects to the local vault folder on this computer.

There is no import. Horizon does not create a second vault, copy your notes, or merge folders.

1. Set up Obsidian Sync

  1. Install Obsidian.
  2. In the vault switcher, choose Open vault from Obsidian Sync > Setup.
  3. Sign in, connect the remote vault, and create its local vault on this computer.
  4. Wait until Obsidian shows the green check and Fully Synced.
  5. Open the vault once and make sure your notes are there.

Obsidian calls the folder on this computer the local vault. That is the folder Horizon needs. Obsidian Sync setup

2. Install Horizon

  1. Extract the entire installer ZIP.
  2. Double-click Install Horizon.cmd.
  3. On first launch, choose the local vault's top-level folder - the folder containing 00_Index.md, Calendar, Inbox, and Runs.

Horizon remembers this folder on the computer and checks it each time the app starts.

3. Connect integrations

Open Settings > Integrations and connect the services you use. Sign-ins and tokens are saved only on this computer.

  • Obsidian data is already connected through the selected vault.
  • Calendar items, projects, research notes, captures, and vault-backed Horizon settings arrive through Obsidian Sync.
  • Microsoft, Google Drive, Zotero, Codex, and other services need a one-time connection on each computer.

Changing vaults later

Use Settings > Data & Storage > Change vault. Horizon checks the new folder, saves its location, and restarts.

Daily workflow

  1. Capture rough tasks, ideas, links, and papers from Home.
  2. Sweep the queue so each item reaches Calendar, Projects, Research, or Notes.
  3. Choose the next priority and use Focus for a bounded work session.

The full guide lives in Settings > Advanced > How to use Horizon.
