# Horizon Workspace Guide

This folder is the durable, human-readable memory behind Horizon. Horizon is the app; this folder is your data.

## Everyday loop

1. Capture a thought, task, link, or idea.
2. Review the Capture Queue.
3. Approve the useful action.
4. Choose one project or next step.
5. Record what changed before you stop.

## Ground rules

- Start at `00_Index.md`, then use the nearest folder `index.md`.
- Keep raw captures in `Inbox/Captures/` until they have been handled.
- Store dated items in `Calendar/Items/`.
- Keep secrets out of notes.
- Prefer small, targeted edits over new systems.

Help is available inside Horizon under **Settings > Advanced > How to use Horizon**.
