# Your Horizon Workspace

Welcome. This folder is where Horizon keeps your notes and planning data.

- **Calendar/** - dated items and the rolling `Now.md` view.
- **Inbox/** - captures waiting to be sorted.
- **Project Registry/** - one note per active project.
- **Research Papers/** - paper notes and research ideas.
- **Runs/** - compact logs created by Horizon.

Open this folder in Obsidian if you want to browse or edit the Markdown files directly.
