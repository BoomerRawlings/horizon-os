# Security policy

Horizon is an early preview. Please do not include API keys, OAuth tokens, personal notes, or private vault files in an issue.

For a suspected vulnerability, open a GitHub security advisory through the repository's **Security > Advisories > New draft security advisory** flow. Include the affected version, platform, reproduction steps, and impact while keeping sensitive data redacted.

Preview binaries are currently unsigned. Download them only from this repository's Releases page and compare the release source before bypassing an operating-system warning.
